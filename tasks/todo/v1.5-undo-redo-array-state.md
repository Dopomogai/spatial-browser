# React Flow Regression: Undo/Redo Broken

## Description
The Undo and Redo tracking in the top bar no longer works. It was strictly mapped to `undoStack` based on the old TLDraw `spatialWidget` object schema, which didn't transition correctly when we flipped the store into the `nodes: AppNode[]` array structure.

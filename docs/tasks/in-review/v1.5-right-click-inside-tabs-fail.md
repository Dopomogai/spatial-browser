# Right Click Inside Tabs - Fresh Spawn Only
**Issue:** The native `webview.addEventListener('context-menu')` fix only seems to apply to tabs freshly spawned in the current session. Existing tabs or tabs updated via navigation lose the right-click binding.
**Hypothesis:** The `context-menu` listener might be detaching on `did-navigate` or is not attaching properly when a node is hydrating from the Zustand store's initial load versus a brand new node array append. 
**Actionable Fix:** Re-audit `BrowserWidgetNode.tsx`. Move the `context-menu` listener out of the `dom-ready` hook if it's trapped there, or verify `webviewRef.current` attachment lifecycle on load.

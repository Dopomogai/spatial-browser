# Drag Release Lag / Input Latching
**Issue:** When resizing or dragging tabs, the drop action is sometimes laggy or fails to register properly, causing the tab to remain "latched" to the cursor until clicked again.
**Resolution:** Re-examine the mouse event listeners and `onPointerDown`/`onPointerUp` handling within `ReactFlow` and the `BrowserWidgetNode`. Ensure no synthetic events or `stopPropagation` calls are blocking the native browser release event.

# Right Click Inside Tabs (Webview Context Menu)
**Issue:** Right-clicking inside the web content of a tab (e.g., on a link, image, or text) is still not working consistently or at all. The goal is to have all standard browser right-click actions available.
**Resolution:** Investigate the IPC communication between the `<webview>` injection script and the main Electron process for the context menu. Ensure the `window.electron?.ipcRenderer?.send?.()` call in the `webview.executeJavaScript` block is correctly configured and that the main process is successfully building and displaying the native menu.

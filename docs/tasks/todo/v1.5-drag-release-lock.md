# Drag Release Lock Warning
**Issue:** The user reports that when dragging or resizing, the node continues to stick to the mouse cursor even after releasing the mouse button, requiring an extra click to un-latch.
**Resolution:** This is a React Flow `PointerEvents` regression inside `BrowserWidgetNode`. Stop explicitly blocking `onPointerDown`/`onClick` in the background container, and let React Flow manage its native hook states without intercepting the release events.

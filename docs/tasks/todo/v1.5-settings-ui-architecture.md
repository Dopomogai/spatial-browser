# Settings UI Architecture & Top Bar Controls

## What we need
The settings UI is weak and needs a dedicated section to control the Top Bar aesthetic (e.g. "Minimal Mode" vs "Show Tabs").

## Why
The user requested a specific toggle to customize the interface layout exactly to their preference (minimal bubbles vs full tab bar).

## How to do it
Overhaul `SettingsWidgetComponent.tsx` and the `BrowserWidgetNode.tsx` settings panel. Add a clean "Top Navigation Bar" section with a dropdown or toggle specifically controlling `isTopTabBarVisible` and the style config for it.

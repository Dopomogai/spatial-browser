# Task: Top Bar Settings Visibility Configuration

## Problem Statement
The Settings menu component itself is mostly functional, but we are missing the Top Bar global settings toggles that allow the user to show/hide the tabs list or fully customize the minimal top navigation bar logic. The wiring between the settings node's toggles and the actual global UI shell logic is incomplete.

## Action Plan
1. Ensure the `<SettingsWidgetComponent />` is properly mounted and renders without crashing.
2. Hook up the local toggles inside the Settings Widget to globally dispatch or mutate the `useCanvasStore.ts` properties (e.g. `isMinimalHeader`, showing/hiding the tab list).
3. Ensure these settings are saved into the `spatial_workspaces` user profile JSON payload.

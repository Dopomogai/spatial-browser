# Task: Settings UI Rendering Crash

## Action Plan
1. Inspect `src/renderer/src/components/shapes/SettingsWidget/SettingsWidgetComponent.tsx` and ensure it is properly exported and registered in the `nodeTypes` mapping in `SpatialCanvas.tsx`.
2. Inspect the `addSettingsWidget` method in `useCanvasStore.ts`. Ensure it handles coordinate offsets and injects `settingsWidget` exactly as named in the React Flow `nodeTypes` mapping.
3. Hook up the `Settings` icon in `TopTabBar.tsx` to dispatch `spawn-settings-widget` event correctly.
4. Verify the Settings node can be moved, closed, and saved.

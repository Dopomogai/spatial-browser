# V1.5 Action Restoration: Undo/Redo Engine & Top Bar State Recovery

## What we need
Undo (`Cmd+Z`) and Redo (`Cmd+Shift+Z`) are completely broken via keyboard, and the UI buttons are missing from the minimal top bar. We need the keyboard shortcuts to work globally (even when typing inside webviews) and the visual buttons restored to the Top Bar.

## Why
It is impossible to recover from a mistake (like accidentally closing or moving a tab) without a functional undo stack, which is the core selling point of a spatial OS.

## How to do it
1. Restore the `<Undo />` and `<Redo />` UI buttons to `TopTabBar.tsx` alongside the Search and Settings icons. 
2. Re-wire `main/index.ts` to capture `Cmd+Z` and `Cmd+Shift+Z`. The previous patch captured the keys but failed to route the IPC command payload correctly into the Zustand store, leaving the app deaf to the keystrokes. Check `App.tsx` global listener `window.electron.ipcRenderer.on('undo-action')` to ensure it successfully calls `useCanvasStore.getState().undo()`.

# Webview Context Menus (Natively)
**Issue:** Right clicking on a webpage inside a tab doesn't spawn the native Copy/Paste/Reload context menu because `window.electron` is undefined in the `webview` execution sandbox. Webviews don't inherently inherit the `preload` script from the main browser window without explicit instructions.
**Resolution:** Fix the `webview.executeJavaScript` to emit an internal HTML message, and catch it in the `<webview>` component wrapper, THEN pipe it to the main process via `window.electron.ipcRenderer`.

# Context Menu Spawn Coordinates
**Issue:** Right clicking on the React Flow canvas to "Spawn Tab Here" isn't functioning correctly. 
**Resolution:** `onPaneContextMenu` intercepts the screen mouse coordinates natively, however, `setOmnibarOpen` expects them translated or expects the `handleSpawnCenter` logic to handle the viewport offset natively. Refactor `Omnibar.tsx` and `SpatialCanvas.tsx` to handle spawning explicitly based on the mouse payload.

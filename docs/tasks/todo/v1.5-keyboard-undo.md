# Keyboard Undo/Redo/Cut
**Issue:** Undo requires `Cmd+Z`, Redo requires `Cmd+Shift+Z` or `Cmd+Y`. And we need a Cut tool mapping (`Cmd+X`).
**Resolution:** Hook global `useEffect` keyboard listeners into `App.tsx` or `SpatialCanvas.tsx` to explicitly bind these directly to the Zustand `undo` and `redo` functions, managing the `e.preventDefault()`. Check for `e.shiftKey` or `e.code === KeyY` explicitly for Redo.

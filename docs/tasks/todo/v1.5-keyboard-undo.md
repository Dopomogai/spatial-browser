# Keyboard Undo/Redo Engine Triggers
**Issue:** `Cmd+Z` and `Cmd+Shift+Z` are missing. The engine operates flawlessly via UI clicks.
**Resolution:** Hook global `useEffect` keyboard listeners checking `e.metaKey`/`e.ctrlKey && e.key === 'z'` into `App.tsx` or `SpatialCanvas.tsx` to bind directly to the Zustand `undo` and `redo` functions.

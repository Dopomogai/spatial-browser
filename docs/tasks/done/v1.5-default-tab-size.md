# Default Tab Size Setting
**Issue:** When spawning a new tab from the Omnibar or Canvas Context Menu, tabs open at a hardcoded default size (currently w: 800, h: 600 or similar). The user wants to specify their preferred default tab size in the settings panel.
**Resolution:** 
1. Add `defaultTabWidth` and `defaultTabHeight` to `useCanvasStore.ts` with sensible defaults (e.g., 800x600).
2. Update the Settings UI (`SettingsWidgetComponent.tsx` and `BrowserWidgetNode.tsx` settings panel) to include two number inputs or a dropdown preset (e.g., "Mobile (400x800)", "Desktop (1024x768)", "Custom") to configure these state values.
3. Update `addWidget` or the `spawn-tab-center` logic to leverage `state.defaultTabWidth` and `state.defaultTabHeight` when declaring the new `AppNode`.

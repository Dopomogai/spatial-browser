# ReferenceError: id is not defined
**Issue:** User reported `Uncaught ReferenceError: id is not defined at handleSpawnSettings (SpatialCanvas.tsx:162:13)`.
**Resolution:** Look at `handleSpawnSettings` inside `SpatialCanvas.tsx`. It tries to construct `id,` in the `AppNode` but `const id = "settings_widget_xxx"` has been deleted or not defined in that function scope. Restore `const id = \`settings_widget_${Date.now()}\`` to `handleSpawnSettings`.
